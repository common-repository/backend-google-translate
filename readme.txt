== Frequently Asked Questions ==

= How does Backend Google Translate work? =

Backend Google Translate allows you to translate the WordPress backend interface into any language with just one click. It also automatically adjusts based on your browser's language settings.

= Which languages does Backend Google Translate support? =

Backend Google Translate supports translation into all languages provided by Google Translate, offering extensive linguistic versatility.

= Can I revert the translation back to the original language? =

Yes, you can easily revert the translation back to the original language by toggling the translation feature off or selecting the original language from the language options.

= Does Backend Google Translate affect the front-end display of my website? =

No, Backend Google Translate is specifically designed to translate only the WordPress backend interface and does not impact the front-end display or content of your website.

= Is there a limit to the number of languages I can translate the backend into? =

No, there is no limit. You can translate the WordPress backend into any language supported by Google Translate, ensuring flexibility and customization according to your preferences.


== Installation ==
1. Download the zip folder named `wp-backend-gtranslate.zip`
2. Unzip the folder and put it in the plugins directory of your wordpress installation. (`wp-content/plugins`).
3. Activate the plugin through the plugin window in the admin panel. You're Done!
